# Font Assets

## Required Fonts

### Orbitron (Headers)
Download from: https://fonts.google.com/specimen/Orbitron

Required files:
- `Orbitron-Regular.ttf`
- `Orbitron-Bold.ttf`

Place these files in this directory.

### Inter (Body Text)
Will be loaded via `google_fonts` package - no manual download needed.

### JetBrains Mono (Code)
System monospace font will be used as fallback.
