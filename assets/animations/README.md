# Animation Assets

Lottie animations and other animation resources for the mascot (Spark) and UI effects.

## Mascot Animations

### Spark States
- `wave.json` - Launch greeting
- `peek.json` - Idle attention-getter
- `celebration.json` - Module completion
- `thinking.json` - Loading state
- `pointing.json` - Onboarding guidance

## Guidelines

- Use Lottie JSON format for vector animations
- Keep animation durations < 3 seconds
- Optimize for 60fps playback
- Test on low-end devices

## Resources

- Create animations at: https://lottiefiles.com
- Export from After Effects with Bodymovin plugin
