# Icon Assets

Custom SVG icons for the Cyber-Fluent design system.

## Custom Icons

Place custom icon assets here that aren't available in Material Icons.

## Format

- Use SVG format for scalability
- Follow 24x24dp base size
- Use single color with opacity for themes
- Name convention: `ic_[name].svg`

## Integration

Convert SVG to Flutter code:
```bash
flutter pub run flutter_svg:avd2flutter assets/icons/
```
